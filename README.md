# PROJECT DINO GAME AI USING GENETIC ALGORITHM

-To clone project:
```
git clone https://github.com/datvodinh10/dino-ai-project-intro-ai-hust.git
```

-To run code:
```
python main.py "num_generation" "num_dino_each_gen"
```
Example:
```
python main.py 10 100
```
-Requirement:
```
pip install pygame
```
```
pip install numpy
```
